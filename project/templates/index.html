About This Tool

This tool leverages server-side logic (as defined in app.py) to handle image conversions, icon generation, filtering, and more. You can:

  • Convert images to a 1024x1024 PNG suitable for use as an Xcode app icon.
  • Generate a full set of iOS app icons from a single image upload.
  • Apply various filters (like grayscale or blur) to your images.
  • Create homescreen mockups and launch screen images.
  • Convert color profiles and generate typography previews.

All actions are performed by the server-side code. Simply select the desired operation, upload your image (if required), and let the backend do the rest.
